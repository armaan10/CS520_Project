# Theta Analysis

This folder contains exploratory data analysis, FCI logs processing, and causal discovery experiments for the Theta project.

## Contents
- `eda_plots_final.py` – generates EDA visualizations
- `fci_logs_2018_q4.py` – processes FCI log data for 2018 Q4
- `run_notears_2018.py` – runs NOTEARs causal discovery
- `results/` – saved figures, graphs, and CSV outputs
